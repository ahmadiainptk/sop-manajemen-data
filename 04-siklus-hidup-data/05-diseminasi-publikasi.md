# Diseminasi dan Publikasi Data

Diseminasi data merupakan proses penyebarluasan informasi yang telah divalidasi kepada publik, sivitas akademika, atau pihak ketiga sesuai dengan tingkat klasifikasi keamanannya.

## Ketentuan Publikasi Portal Data:
1. **Portal Data Institusi:** IAIN Pontianak wajib memiliki satu gerbang akses informasi resmi berupa "Portal Data Institusi" yang dapat diakses publik untuk melihat kumpulan agregat data non-rahasia (seperti jumlah mahasiswa aktif per fakultas atau daftar program studi yang terakreditasi).
2. **Open Data API:** Selain disajikan dalam bentuk grafis atau tabel di portal web, data publik juga harus disediakan dalam format *machine-readable* melalui titik akhir (*endpoint*) *Open Data API* guna memudahkan peneliti atau pihak luar yang membutuhkan data agregat kampus.
3. **Penyaringan Data Sensitif:** Sebelum sebuah *dataset* dipublikasikan secara umum, *Walidata* wajib memastikan bahwa data tersebut telah melalui proses de-identifikasi (penghapusan informasi spesifik seperti NIK, nomor telepon, atau alamat detail) agar tidak melanggar ketentuan privasi perlindungan data pribadi.
